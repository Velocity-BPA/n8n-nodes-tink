# Licensing FAQ

## General Questions

### What license is this project under?
This project is licensed under the Business Source License 1.1 (BSL 1.1).

### Is this open source?
BSL 1.1 is a source-available license. The source code is freely viewable and usable for non-commercial purposes, but commercial use requires a separate license.

### When does this become fully open source?
On January 1, 2030, this project will automatically convert to the Apache License 2.0.

## Use Cases

### Is internal SaaS infrastructure considered internal use?
No. If the infrastructure supports a revenue-generating service, it is considered commercial use.

### What if customers never see the node?
Visibility is irrelevant. Revenue enablement constitutes commercial use.

### What if we are a startup?
Startups are commercial entities and require a commercial license for production use.

### What if we only use one workflow?
Scope does not change license requirements. Any commercial use requires a license.

### Do proof-of-concepts require a license?
Internal, non-customer-facing POCs are permitted. Production SaaS deployments require a license.

### Can I use this for personal projects?
Yes, personal use is permitted under the BSL 1.1.

### Can I use this for educational purposes?
Yes, educational use is permitted under the BSL 1.1.

### Can I use this for research?
Yes, research use is permitted under the BSL 1.1.

### Can I use this internally at my company?
Yes, internal business use is permitted, provided it does not involve resale, redistribution, or offering the Licensed Work as part of a commercial product or service.

### Can I modify the source code?
Yes, you can modify the source code for permitted uses. Commercial use of modified code still requires a commercial license.

### Can I contribute to the project?
Yes, contributions are welcome. By contributing, you agree to license your contributions under the same BSL 1.1 terms.

### Can I fork this project?
Yes, you can fork the project, but your fork remains under the same BSL 1.1 license terms.
